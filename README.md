## 🚀 Como executar o projeto

1. Abra o terminal na pasta do projeto
2. Execute os comandos:

```
dotnet restore
dotnet run
```

3. Acesse no navegador:

```
http://localhost:5171
```

* C#

## 🖥️ Sobre o projeto

Este projeto consiste em um currículo online desenvolvido em Blazor, com foco em apresentar informações pessoais, experiência, habilidades e formação de forma organizada e intuitiva.

O layout foi desenvolvido com tema escuro, visando melhor experiência visual.

---



## 🎯 Heurísticas de Usabilidade

* Interface simples e fácil de navegar
* Informações organizadas em seções
* Layout limpo e legível
* Boa escolha de cores (modo escuro)
